# Challenge Rules

Esta base segue apenas regras genericas para o passo inicial:

- O agente roda em modo `dry-run`.
- Nenhuma acao externa real deve ser executada.
- O provedor de LLM padrao e `mock`.
- Gemini/OpenAI ainda nao devem ser implementados.
- O resultado da execucao deve ser salvo em `runs/<timestamp>/`.
- O projeto nao usa frontend nem banco de dados.
